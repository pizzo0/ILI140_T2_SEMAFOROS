# ILI140 - T2

## Como usar

Con traffic-sim abierto en Greenfoot, se pueden generar vehiculos de la siguiente forma:

- "w" para generar por arriba
- "a" para generar por la izquierda
- "s" para generar por abajo
- "d" para generar por la derecha
- "r" genera vehiculos de forma aleatoria

## Requerimientos

Es necesario:

- Greenfoot
- activeMQ
